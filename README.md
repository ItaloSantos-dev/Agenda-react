# Agenda em React

Aplicação simples para gerenciamento de contatos usando React, TypeScript e DexieJS.


## Tecnologias
- React
- TypeScript
- Bootstrap
- Dexie.js (IndexedDB)


## Funcionalidades
- Cadastrar contatos
- Editar contatos
- Excluir contatos
- Persistência local com IndexedDB
